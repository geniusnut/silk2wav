# silk2wav
Convert Wechat Silk to wav file. 

```shell
.\silk2wav.exe
Usage: silk2wav.exe [OPTIONS] [INPUT_FILE]

Arguments:
  [INPUT_FILE]

Options:
  -s, --sample-rate <SAMPLE_RATE>  [default: 16000] [possible values: 8000, 16000, 441000]
  -h, --help                       Print help
  -V, --version                    Print version
```
